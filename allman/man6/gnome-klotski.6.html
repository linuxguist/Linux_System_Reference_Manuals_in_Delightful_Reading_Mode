NAME

Klotski - Sliding block puzzles game for GNOME

SYNOPSIS

gnome-klotski [OPTION...]

DESCRIPTION

gnome-klotski is a clone of the Klotski game. The objective is to move the patterned block to the area bordered by green markers.

OPTIONS

-v, --version

Print release version and exit

This program also accepts the standard GNOME and GTK options.
