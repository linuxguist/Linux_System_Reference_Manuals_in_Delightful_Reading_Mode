NAME

Date::Language - Language specific date formating and parsing

SYNOPSIS

use Date::Language; my $lang = Date::Language->new(German); $lang->time2str("%a %b %e %T %Y\n", time);

DESCRIPTION

Date::Language provides objects to parse and format dates for specific languages. Available languages are

Afar French Russian_cp1251 Amharic Gedeo Russian_koi8r Austrian German Sidama Brazilian Greek Somali Chinese Hungarian Spanish Chinese_GB Icelandic Swedish Czech Italian Tigrinya Danish Norwegian TigrinyaEritrean Dutch Oromo TigrinyaEthiopian English Romanian Turkish Finnish Russian Bulgarian Occitan
