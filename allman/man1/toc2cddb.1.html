NAME

toc2cddb - translates a TOC file of cdrdao(1) into a cddb file and prints it to stdout

SYNOPSIS

toc2cddb toc_file

toc2cddb [ -h | -V ]

DESCRIPTION

toc2cddb translates a TOC file of cdrdao(1) into a cddb file and prints it to stdout.

OPTIONS

-h

Shows a short help message.

-V

Prints the version of toc2cddb.

SEE ALSO

cdrdao(1)
